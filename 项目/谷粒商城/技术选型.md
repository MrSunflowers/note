# 环境准备

参照笔记安装

- CentOS 7 64 位
- docker
- docker-mysql:5.7
- docker-redis
- node.js
- npm

[下载人人开源项目](https://gitee.com/renrenio)

renren-fast-vue 前端项目 
```
npm install
npm run dev
```
renren-fast 后台项目 修改配置文件的数据库地址 直接启动

renren-generator 代码生成器 (可选)

技术选型： 

nacos 注册中心，配置中心
ribbon 负载均衡
openFeign 远程调用
sentinel 服务容错
gateWay API 网关
sleuth 调用链路监控
seata 分布式事务管理
